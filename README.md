# opengl-robot-animation
## 프로젝트 개요
본 프로젝트는 컴퓨터 그래픽스 수업에서 진행한 기본 실습을 확장하여,
정적인 로봇 모델을 실제 상호작용 애니메이션으로 발전시킨 개인 프로젝트입니다.

## 수업 실습 범위
- 정육면체 기반 3D 오브젝트 렌더링
- 큐브 조합을 통한 로봇 모델 구성
- 기본 회전 애니메이션 구현
- Depth Buffer 적용 및 기본 조명 처리

## 개인 확장 구현
- 텍스처 매핑을 추가하여 시각적 완성도 향상
- 사운드 효과를 연동해 상호작용 피드백 구현
- 배경 오브젝트 및 공간 구성
- 로봇이 흙 블록을 캐고, 쌓고, 그 위로 이동하는 애니메이션 설계
- 상태 기반 애니메이션 로직 구성

  
※ 본 프로젝트는 OpenGL/GLUT 기반으로 실행 환경 구성이 복잡하여,
실행보다는 구현 방식과 코드 구조를 중심으로 공유합니다.
